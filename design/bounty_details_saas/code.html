Optimize JIT compiler performance for large enterprise projects

Reward $850.00
Est. Time 12-16 hrs
Difficulty Expert

description Description

Detailed technical requirements for performance optimization in Tailwind's JIT engine. We've observed linear degradation in build times as enterprise projects scale beyond 5,000 components using complex variants.

The goal is to implement a more aggressive caching strategy and parallelize the scanning of utility classes across multiple CPU threads. Specifically, the candidate should focus on the following areas:

  • Improve the memoization of generated CSS for static class patterns.
  • Optimize the AST walking process for React/Next.js large-scale projects.
  • Reduce peak memory usage during initial watch-mode scans.
